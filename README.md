# Burlington Eastside Automotive — Website

A 3-page website for **Burlington Eastside Automotive**, an auto repair & maintenance shop in Burlington, ON. Built by **Apex Web Studio** as a free sample.

- **Live:** https://anmtal.github.io/Burlington-Eastside-Automotive/
- **Facebook:** https://www.facebook.com/p/Burlington-East-Side-Automotive-100069459742204/

## Pages
- `index.html` — hero, services, how-it-works, about, trust stats, work gallery + lightbox, reviews, hours & location.
- `services.html` — full price guide (maintenance, brakes & tires, repairs & diagnostics, inspections).
- `book.html` — multi-step service-request form with live estimate, FAQ, contact.

## Stack
Static HTML/CSS/JS — no build. **Trust-blue** theme (design system via the ui-ux-pro-max skill: primary #2563EB, accent amber #D97706), Oswald + Inter. **Framer-Motion-style animation via Motion** (`motion@12.42.2`, vanilla, loaded from CDN in `js/motion.js`) — spring reveals, scroll-linked hero zoom, tactile hover/press. Progressive enhancement: if the Motion CDN is unavailable, the CSS + `main.js` reveal system still shows everything.
